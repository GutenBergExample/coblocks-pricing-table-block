GutenKit is an upcoming collection of Gutenberg blocks that let you easily add pricing tables to any landing or sales page using Gutenberg.

== Description ==

[GutenKit](https://gutenkit.com?utm_medium=gutenkit-pricing-tables&utm_source=wordpressorg&utm_campaign=readme&utm_content=gutenkit) is an upcoming suite of Gutenberg Blocks designed to help you build better landing and sales pages within Gutenberg.

This plugin is created and maintaned by [Rich Tabor](https://richtabor.com?utm_medium=gutenkit-pricing-tables&utm_source=wordpressorg&utm_campaign=readme&utm_content=rich-tabor).

== Installation ==

1. Upload the `pricing-table-block-gutenberg` folder to your `/wp-content/plugins/` directory or alternatively upload the pricing-table-block-gutenberg.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your local computer.
2. Install and activate the Gutenberg WordPress plugin.
2. Activate the plugins through the 'Plugins' menu in WordPress.
3. Add pricing tables within Gutenberg with the new "Pricing" blocks.
